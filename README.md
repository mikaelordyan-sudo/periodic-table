# 🧪 Periodic Table 3D
Это крутой проект с визуализацией таблицы.

## 🚀 Функции
* Переключение между режимами: Table, Sphere, Helix, Grid.
* Плавная анимация с использованием библиотеки GSAP.
* Интерактивное вращение мышкой.

## 🛠 Технологии
* HTML5 / CSS3 (Glassmorphism)
* JavaScript (ES6 Modules)
* GSAP (GreenSock Animation Platform)

## 📦 Как запустить
1. Склонируйте репозиторий.
2. Откройте `index.html` через Live Server в VS Code.
